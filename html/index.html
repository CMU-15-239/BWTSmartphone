Word Definition Edit

Info

About

This app lets you edit the lessons and assignments tied to the hangman game for the BWT.

To edit any word or definition, simply click on it. To add a word, simply click the "Add word" button below the table.

To save any changes you've made to a word, or to delete a word, click the appropriate button at the end of the word's row.

Notes

Words cannot be more than 8 characters long. If you try to add a word longer than 8 characters, its row will be shaded red to alert you.

If a save or a delete succeeds, its row will shade green briefly. If it fails, the row will shade red until a save or delete is re-attempted.
